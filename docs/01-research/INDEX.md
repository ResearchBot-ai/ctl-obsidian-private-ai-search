# Research Index

## Directory Purpose

This directory contains research artifacts, prototypes, and experimental findings organized by release lineage.

## Quick Navigation

### Research Releases

Research is organized by release:

- Release-1-GenAI-Assets/ - First release research materials
- Release-2-GenAI-Assets/ - Second release research materials

### Research Categories

Within each release directory, research is organized by type:

- **models/** - AI/ML models and specifications
- **benchmarks/** - Performance benchmarks and comparisons
- **findings.md** - Research conclusions and insights

## Adding New Research

To add new research artifacts:

1. Determine the associated release
2. Create or navigate to the release directory
3. Add artifacts in the appropriate subdirectory
4. Update findings.md with research conclusions
